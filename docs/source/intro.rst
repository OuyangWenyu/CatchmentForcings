Introduction
============================
This is a dataset zoo for watershed hydrological modeling, especially with data-driven methods.

Overview
--------------------

Now we mainly download and process the following datasets

- CAMELS/MOPEX/LAMAH
- Daymet
- ECMWF
- MODIS
- NLDAS
